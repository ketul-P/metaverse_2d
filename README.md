## 2D Metaverse 
### 🎮🕹️👾

A browser-based 2D virtual world where users can create avatars, move around shared spaces, and interact with other users in real time.

```
Please note that this project is still in initial phases.
```

## Overview / Ideas

This project is a 2D multiplayer virtual space where users can:

- Join shared rooms/maps as customizable avatars
- Move around in real time using arrow keys / WASD
- See and interact with other connected users
- Chat or communicate with nearby users (proximity-based or global)
